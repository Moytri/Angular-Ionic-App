This is my BCIT homework 04. It reads and displays data asynchronously from a JSON file. Allows a user to store and read data using Ionic storage. It uses ionic components.

  • "Home" tab reads data from Ionic storage. Click "Menu" icon from Home nav.
  • "About" tab shows icons and details.
  • "Contact" tab reads contact list from local JSON file, "fab-icon" is included at right bottom.
  • "More" tab works as Ionic storage to store and read data!
  • "Film" tab reads film name from remote source!
